# IntelliToggle Homepage

A creation of the IntelliToggle homepage design built with **Nuxt 3**, **Vue 3**, and **TailwindCSS**.

## Hosted Link 

https://intellitoggle-home.vercel.app/

## Stack

- **Nuxt 3** 
- **Vue 3** 
- **TailwindCSS** 
- **Vite** 

## Interactive Element

**Mobile Menu Toggle** (Header component)
   - Hamburger icon that transforms to X
   - Smooth slide-down animation
   - Accessible with proper ARIA labels

### Tradeoffs & Shortcuts
   - Did not optimize fully for mobile
   - Did not fully ensure matching font/text size in some areas
   - Skipped putting the background image in the section before the footer
